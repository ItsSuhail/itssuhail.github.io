Welcome to Electroid! ⚡

Electroid simulates the motion of electric charges in a 2D space under electrostatic forces (and optionally gravity).

Core Features:

  • Infinite Canvas: Right-click drag, Middle-click drag, or Space + Left-click drag to pan. Scroll wheel to zoom centered on cursor.
  • Interactive Manipulation: Pause the simulation and drag charges around directly on the canvas to set their starting positions.
  • Detailed Inspector: Click on any charge to select it. You can modify its label, charge, mass, initial velocity, or toggle whether it is held (anchored in place).
  • Boundary Box (Red dashed line): If a particle leaves this box, it won't be rendered. However, it still exerts force on other charges. If all charges leave the boundary box, the simulation terminates.

Vector Indicators:

  • Green Vector: Velocity  v
  • Orange Vector: Net Force  F
